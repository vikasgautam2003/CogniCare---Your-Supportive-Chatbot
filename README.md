# 🏥 CogniCare - AI-Powered Healthcare Chatbot

CogniCare is a **healthcare chatbot platform** built with **Next.js**, **Tailwind CSS**, and integrated with the **Gemini AI API**. It provides intelligent healthcare responses, secure user authentication, and seamless email notifications via EmailJS.

## 📖 About The Project

CogniCare is a **Gemini AI-powered healthcare chatbot** that assists users with medical inquiries, appointment guidance, and healthcare advice. It includes **secure login and registration**, persistent user data storage, and a responsive, modern interface.

![Project Screenshot](public/cogni1.png)

---

## 🌟 Key Features

- **AI-Powered Chatbot**: Provides healthcare guidance using Gemini API.  
- **Secure Authentication**: Uses JWT and bcrypt for password hashing and session management.  
- **Email Notifications**: Users receive emails via EmailJS for registration and appointment confirmations.  
- **Responsive Design**: Built with Tailwind CSS for mobile-first, modern UI.  
- **Next.js Optimizations**: Server-side rendering and fast performance.

---

## 🔧 Tech Stack

| Technology      | Role & Justification                                                                 |
| :-------------- | :--------------------------------------------------------------------------------- |
| **Next.js**     | React framework with SSR and optimized performance.                                 |
| **Tailwind CSS**| Rapid and responsive UI styling with utility-first classes.                         |
| **Gemini API**  | AI backend providing healthcare responses to user queries.                         |
| **JWT**         | Secure user authentication and session management.                                  |
| **bcrypt**      | Password hashing for secure storage of user credentials.                            |
| **EmailJS**     | Sends emails directly from frontend without requiring a backend mail server.       |
| **JavaScript/TypeScript** | Core frontend and backend programming language.                            |
| **Vercel**      | Hosting platform for live deployment.                                              |
